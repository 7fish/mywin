Troubleshooting Tips

  • Ensure both services are running on ports 3000 and 3001
  • Check that your firewall isn't blocking these ports
  • For HTTPS, make sure you have a valid certificate for localhost
  • Try accessing the endpoints directly in your browser
  • Restart your services if they're not responding
  • If using a React/Next.js dev server, check the terminal for errors
